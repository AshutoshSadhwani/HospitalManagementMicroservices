# Hospital Management System - Microservices Architecture

## Project Overview

This project is a Hospital Management System developed using Java Spring Boot Microservices Architecture.

The system includes:

- Patient Service
- Doctor Service
- Room Service
- Billing Service
- Eureka Service Registry
  

## Technologies Used

- Java 21
- Spring Boot
- Eureka Server
- OpenFeign
- PostgreSQL
- Hibernate/JPA
- Maven
- Postman - for api testing

## Features

- Add patients
- Doctor management
- Automatic room allocation
- Billing display (not added payment gateway yet)
- Service discovery using Eureka
- Inter-service communication using Feign Client
